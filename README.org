* Generate codegouvfr data

"codegouvfr" stands for [[https://code.etalab.gouv.fr]].

This repository helps creating =json= files used by codegouvfr frontend.

* Generate the binary file

Assuming GraalVM >19.3 is installed:

: clj -A:native-image
